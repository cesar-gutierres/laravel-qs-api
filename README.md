# laravel-qs-api

Classe simples para tratar de requisições GET para consultas elegantes no eloquent.

## Termos aceitos

- where
- whereIn
- whereNotIn
- whereNull
- whereBetween
- whereNotBetween
- whereTime
- whereDate
- whereDay
- whereMonth
- whereYear
- orWhere
- select
- paginate
- limit
- offset
- scope
- bind
- with
- orderBy
